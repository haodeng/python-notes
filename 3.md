Output input


 formatted string literals, begin a string with f or F before the opening quotation mark or triple quotation mark.
```python
>>> first_name = 'Hao'
>>> last_name = 'Deng'
>>> f'full name {first_name} {last_name}'
'full name Hao Deng'
>>> f'''Full name
... {first_name} {last_name}'''
'Full name\nHao Deng'
```
